Ska Radios of the World, Unite!

When I started File Under Ska in 2003, people could only hear it via FM in Stuttgart (which has a challenging topology).
In the following years, the Freies Radio für Stuttgart and other free radios came up with internet livestreams, reaching listeners all over the world.
Timezones (with daylight saving time and half-hour steps) are really tricky to implement. So there is no warranty for correctness... I've had this idea for a long time and reserved the domain when it came out.
So this is one of the projects that AI helped finish.
I am a software engineer by trade and I've used HTML since the mid-90s, but I am not a web designer. Help with UI is welcome.
